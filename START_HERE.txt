Problem: "npm: command not found"
Solution: You need to install Node.js first
         Download from: https://nodejs.org/

Problem: "Port 5000 is already in use"
Solution: Another application is using port 5000
         Either close that application or change the port
         in server/index.ts

Problem: Installation is taking too long
Solution: This is normal! Installing can take 5-10 minutes
         depending on your internet speed

Problem: Build errors
Solution: Delete node_modules folder and try again:
         rm -rf node_modules package-lock.json
         npm install
